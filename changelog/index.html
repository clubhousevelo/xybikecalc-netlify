▼ Changelog

A detailed history of updates and changes to XY Bike Calc.

Version 1.6.1

September 17, 2025
  • Updated the database and position calculators to incorporate stock cockpit and max spacer values when provided by the manufacturer. These values are highlighted in yellow in the XY Position Calculator and Bike Position Simulator
  • Added the ability to add a second bike to Bike Position Simulator to overlay a second bike/position for visual comparison
  • Added "Frame Material" as a filterable option in the Bike Search
  • Corrected BB to Rail calculation in XY Position Calculator (was calculating BB to SRC)
  • Added a section for Product Manuals currently supplied by manufacturers for easy reference

Version 1.6

August 20, 2025
  • Created new "Bike Position Simulator" to help users visualize both saddle and handlebar positions. This calculates and displays Cartesian and polar coordinates of the frame, saddle, and handlebar
  • Added "Headset Height" value inputs to XY Position Calculator, Stack & Reach Calculator, and Bike Position Simulator to differentiate between spacers and headset/transition spacer stack
  • Updated "Bike Search" with sliders, brand filter, and Stack/Reach Ratio filter. Added Stack/Reach Ratio to the results table. Bikes can now be added to XY Position Calculator directly from the Bike Search results table

Version 1.5.2

July 31, 2025
  • Fixed a bug in the XY Position Calculator visual where 0° stem angle was treated as no value and defaulted to -6°
  • Redrew the saddle image in the Seatpost Calculator to be shorter in height
  • Added "BB to Saddle XY" calculation to the Seatpost Calculator to differentiate it from "BB to Rail", which is measured along the STA

Version 1.5.1

July 30, 2025
  • Added a "Notes" section to each bike configuration in the XY Position Calculator to allow for additional notes about that particular bike. This will be saved to client profiles and will show up in the Print document

Version 1.5

July 28, 2025
  • Created a new separate Seatpost Calculator with a visual aid to help calculate seatpost length, angle, and other values
  • Renamed "HX/HY to Frame Calculator" to "Stack & Reach Calculator" and updated the calculator to include sliders for input value adjustment and added a visual aid to the calculator

Version 1.4.2

July 22, 2025
  • Added a visual aid to the XY Position Calculator to help visualize and comparethe calculated handlebar positions
  • Added a 'Donate' page to support the site 🙏

Version 1.4.1

July 7, 2025
  • Restructured site directories and created a new start page

Version 1.4

March 20, 2025
  • Moved the rider information section from the top to the left column of the XY Position Calculator. Added a toggle to collapse/expand the section to create more real estate for bike comparisons
  • Added a "Component Notes" section to the client profiles to note other details about the equipment used or planned to be used
  • Added hover text to XY Position Calculator, Stem Calculator, and Stack & Reach Calculator to explain what or how values are calculated/measured
  • Reformatted the Print Results page to include this new component notes section
  • Resized and repositioned stem comparison visual in Stem Calculator for easier viewing and to prevent deformation of the graph on mobile devices

Version 1.3

March 14, 2025
  • Added Terms of Service
  • Required acknowledgement of Terms of Service and Privacy Policy during account creation
  • Login is now required to access XY Position Calculator database
  • Added "Notes" section to client profiles to note other details about the client or equipment
  • Added a visual aid to the Stem Calculator to more easily visualize stem comparisons

Version 1.2.1

March 9, 2025
  • Added "Guide" page to explain how to use the XY Position Calculator
  • Reformatted printed results page to more closely reflect the XY Position Calculator
  • Improved navigation bar code consistency in desktop and mobile views

  • Added Google Analytics tracking
